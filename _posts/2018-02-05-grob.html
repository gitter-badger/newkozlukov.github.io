I'm planning an effort to try and translate some songs and poems of Yegor Letov, GrOb, and Yanka. I'm probably going to have to find new rhymes which would still translate this feeling of an utmost despair and detachment. Hence I'll begin with a table of just some chants and words.

Motivation

We often debate whether it makes sense to save Russian language. Not that we want to eliminate it. It's rather a joke. Yet in spite of that we strive to think seriously. And we find no convincing reason for (any) ethnic language to be. In these discussions my mockingly argument for saving the language is that you can only sing Grazhdanskaya Oborona in Russian. One could say it's all nonsense but I'm still going to test this argument: I'll try to translate some Yegor Letov's poems.
